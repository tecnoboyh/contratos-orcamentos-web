# Contratos & Orçamentos Web

Frontend em React + Vite + TailwindCSS para o sistema de controle de contratos e gestão orçamentária.

## Rodar localmente

```bash
npm install
cp .env.example .env
npm run dev
```

Por padrão, o frontend usa a API em:

```env
VITE_API_URL="http://localhost:3333/api"
```

## Telas iniciais

- Login
- Cadastro
- Dashboard
- Contratos
- Novo contrato
- Assinaturas
- Obras
